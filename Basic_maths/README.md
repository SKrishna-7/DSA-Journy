# 🧮 Basic Maths in C++ — DSA Foundation

This folder contains fundamental **mathematical algorithms** that form the base of most **Data Structures and Algorithms** concepts.  
These problems strengthen logic, number manipulation, and pattern recognition — all of which are essential before diving into arrays, recursion, and number theory.

---

## 🧩 Topics Covered

| File Name | Concept | Description | Time Complexity |
|------------|----------|--------------|------------------|
| `Digits.cpp` | Count Digits | Counts the number of digits in an integer using both loop and logarithmic method. | O(log₁₀N) |
| `revnumber.cpp` | Reverse a Number | Reverses digits of a number using modulus and division operations. | O(log₁₀N) |
| `palindrome.cpp` | Palindrome Check | Checks if a number reads the same backward and forward. | O(log₁₀N) |
| `amstrong.cpp` | Armstrong Number | Verifies if a number equals the sum of cubes of its digits (e.g., 153). | O(log₁₀N) |
| `print_all_divisors.cpp` | Print All Divisors | Efficiently finds all divisors of `n` using the √n optimization. | O(√N) |
| `gcd_hcf.cpp` | GCD / HCF | Finds Greatest Common Divisor using **Euclidean Algorithm** (iterative & recursive). | O(log(min(a, b))) |
| `prime.cpp` | Prime Number Check | Determines if a number is prime using trial division up to √n. | O(√N) |
